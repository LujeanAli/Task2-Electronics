# Task2-Electronics


## Servo Motor

 ### Definition:
  A Servo Motor is a small device that has an output shaft. This shaft can be positioned to specific angular positions.

 ### Hardware Required:
  - Arduino UNO.
  - Breadboard.
  - Servo Motor.
  - 10k ohm potentiometer
  - hook-up wires.
